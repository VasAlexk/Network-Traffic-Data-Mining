# Network-Traffic-Data-Mining

# Data Mining & Machine Learning: Network Traffic Analysis (2025)

## Περιγραφή
Εξόρυξη δεδομένων και τη χρήση αλγορίθμων μηχανικής μάθησης για την ανάλυση και ταξινόμηση δικτυακής κίνησης. Στόχος είναι ο εντοπισμός κακόβουλης δραστηριότητας (Malicious) έναντι κανονικής (Benign), καθώς και η ακριβής αναγνώριση επιμέρους τύπων επιθέσεων (Traffic Type).

## Τεχνολογίες & Βιβλιοθήκες
Η υλοποίηση έγινε σε **Python** χρησιμοποιώντας τις εξής βιβλιοθήκες:
* **Προεπεξεργασία & Ανάλυση:** `pandas`, `numpy`, `scipy`
* **Οπτικοποίηση:** `matplotlib`, `seaborn`, `plotly`
* **Machine Learning:** `scikit-learn` (StandardScaler, PCA, MLPClassifier, SVM)
* **Διαχείριση Ανισόρροπων Δεδομένων:** `imblearn` (SMOTE)
* **Clustering:** `hdbscan`, `MiniBatchKMeans`

## Μεθοδολογία & Ροή Εργασίας

### 1. Προκαταρκτική Ανάλυση (EDA)
* Παραγωγή περιγραφικών στατιστικών και έλεγχος για κενές τιμές.
* Οπτικοποίηση δεδομένων με **Histograms**, **Boxplots**, **Violin plots** και **Countplots** (με λογαριθμική κλίμακα λόγω ανισορροπίας των κλάσεων).
* Ανάλυση συσχετίσεων με **Heatmaps** για τον εντοπισμό πλεονάζουσας πληροφορίας.

### 2. Μείωση Διαστατικότητας & Προεπεξεργασία
* **Χειροκίνητη αφαίρεση:** Διαγραφή στηλών με μηδενική διακύμανση ή ανούσια πληροφορία (π.χ. Flow ID, Timestamp).
* **PCA (Principal Component Analysis):** Εφαρμογή για τη διατήρηση του 95% της συνολικής πληροφορίας.
* **Scaling:** Κανονικοποίηση δεδομένων με `StandardScaler`.

### 3. Διαχείριση Όγκου Δεδομένων & Clustering
Λόγω του μεγάλου όγκου των δεδομένων, χρησιμοποιήθηκαν τρεις διαφορετικές προσεγγίσεις:
* **Stratified Sampling:** Στρωματοποιημένη δειγματοληψία 10% για τη διατήρηση της αναλογίας των κλάσεων.
* **MiniBatchKMeans:** Ομαδοποίηση και επιλογή αντιπροσωπευτικών δειγμάτων σε συνδυασμό με SMOTE.
* **HDBSCAN:** Clustering βασισμένο στην πυκνότητα για τον εντοπισμό μοτίβων και αφαίρεση θορύβου.

## Μοντέλα & Αποτελέσματα
Εκπαιδεύτηκαν και συγκρίθηκαν δύο βασικοί ταξινομητές: **Multi-Layer Perceptron (MLP)** και **Support Vector Machines (SVM)**.

| Μοντέλο | Dataset | Στόχος | F1-Score |
| :--- | :--- | :--- | :--- |
| **MLP** | Stratified Sample | Label (Benign/Malicious) | **0.999** |
| **MLP** | Stratified Sample | Traffic Type | **0.998** |
| **SVM** | Stratified Sample | Label (Benign/Malicious) | **0.945** |
| **SVM** | Stratified Sample | Traffic Type | **0.800** |

### Βασικά Συμπεράσματα
* Το μοντέλο **MLP (Νευρωνικό Δίκτυο)** υπερείχε σημαντικά, επιτυγχάνοντας σχεδόν τέλεια αποτελέσματα στο δειγματοληπτικό σύνολο.
* Το **SVM** ήταν αποτελεσματικό στη δυαδική ταξινόμηση (Label), αλλά δυσκολεύτηκε στη διάκριση παρόμοιων τύπων κίνησης (π.χ. Text vs Audio).
